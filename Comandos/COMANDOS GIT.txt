crear el repositorio

git init -> inicializar carpeta como repositorio vacÃ­o

git remote add origin (enlace SSH) 

git status -> ver estado

git fetch origin main

git pull origin main -> enlazar

git add .  -> incluir todos los archivos de la cerpeta.

git status -> ver estado

git commit -m "Primer commit"  -> cargar archivos

git push -u origin main -> subir todo

git push -u origin main --force -> Sube todo sin importar que tenga un error

git config --list  -> ver las configuraciones


----------------------------------------------------------

actualizar

git status -> ver estado

git pull origin main -> refresca por si ya hicieron cambios

git add .  -> incluir todos los archivos de la cerpeta.

git commit -m "se agregaron otras funciones"  -> cargar archivos

git push -u origin main -> subir todo
